FPGA Music Player

This project implements a remote-controlled music player on the FPGA that interfaces with a VGA monitor to display the cover art. The FPGA fabric serves as the main control unit of the music player, and it interfaces with various components on the DE-115 board to process data such as the:

  • Music samples.
  • Cover art image
  • Title image
  • Remote control signals
  • Status icons (Play, Next/Previous Song, Pause, Playback Speed)
This project is entirely hardware based for improved performance and does not rely on a SoC (NIOS II).
